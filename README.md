# MLHousing

MLHousing est un projet d'analyse et de prédiction des prix de logements basé sur des techniques de machine learning.

## Description

Ce dépôt vise à explorer et appliquer différents algorithmes d'apprentissage automatique pour prédire la valeur des maisons en se basant sur des caractéristiques du marché immobilier.

## Structure du dépôt

- `data/` : dossiers et fichiers de jeux de données
- `notebooks/` : notebooks Jupyter pour l'exploration et la modélisation
- `scripts/` : scripts Python destinés au prétraitement et à l'entraînement
- `models/` : modèles sauvegardés et résultats

## Fonctionnalités

- Nettoyage et prétraitement des données
- Exploration et visualisation des données
- Modélisation par différents algorithmes ML (régression, arbres, etc.)
- Évaluation des performances

## Prérequis

- Python 3.x
- Les principales librairies : numpy, pandas, scikit-learn, matplotlib, seaborn
- Jupyter notebook (optionnel)

## Installation

Clonez ce dépôt :

```bash
git clone https://github.com/Bounama2003/MLHousing.git
cd MLHousing
```

Installez les dépendances :

```bash
pip install -r requirements.txt
```

## Utilisation

1. Placez le jeu de données dans le dossier `data/`
2. Lancez le notebook principal depuis le dossier `notebooks/`
3. Suivez les instructions pour entraîner et évaluer les modèles
